# 2024 Race 3 Page Package

Replace this file in your project:

app/media/race-reports/2024-race-3/page.tsx

with the included `page.tsx`.

Required images must already be in:

public/images/race-reports/2024-race-3/

Required image names:
- hero.jpg
- briefing.jpg
- track.jpg
- team.jpg
- engine.jpg
- phone.jpg
- kp-forecast.jpg
- hangar.jpg

After replacing the file, run:

npm run build

Then view:

http://localhost:3000/media/race-reports/2024-race-3
